# JM News

Read fake news articles and look up words in real-time. Actually there is [10ten-ja-reader](https://github.com/birchill/10ten-ja-reader) extension with the same functionality, but it's only work on Firefox if we using it on mobile (Android).

I need something i can access from anywhere, without any extensions. That's why i built this website. The term "JM" is actually comes from [JMDict](https://github.com/scriptin/jmdict-simplified). 

The news is actually AI generated and may not really accurate in term of grammar or anything, but it's acceptable for me.

If you find any bugs, just let 
me know by opening a new issue, and i will look into it.
